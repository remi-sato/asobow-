## サイト概要
### サイトテーマ
犬と利用できる遊び場の情報や口コミを共有し、飼い主同士の交流や犬の遊び相手を探す手助けをするレビュー及びコミュニティサイト
​
### テーマを選んだ理由
我が家にはビーグルの子犬がいます。<br>
犬にとって子犬の時期は「社会化期」と呼ばれ、人や犬、さまざまな環境に慣れるための大切な時期です。そのため、安全に遊びながら他の犬や人と触れ合える場所を探していました。<br>
しかし、実際に調べてみると、実家の近くの大きな公園は犬の立ち入りが禁止されていたり、ドッグランも小型犬専用で利用できなかったりと、犬が自由に遊べる場所は意外と限られていました。また、利用条件や雰囲気などは実際に訪れてみないと分からないことが多く、情報収集に苦労しました。<br>
さらに、私は現在、縁もゆかりもない地に引越してきたばかりで、犬に関する情報を共有できる知人（以下、犬友とする）が周囲にいません。そのため、全国の愛犬家同士が犬と遊べる場所の情報や利用者の感想を共有できるサービスがあれば便利だと考えました。<br>
また、犬友がいないと犬同士を遊ばせる機会も生まれづらいため、地域や犬種、サイズなどをきっかけにコミュニティを作り、飼い主同士が交流できる場も提供したいと考えています。<br>
このサービスは現在犬を飼っている人のみでなく、今後犬を迎えようと考えている人にとっても役立つ情報源になると考え、このテーマをポートフォリオとして制作することにしました。

### ターゲットユーザ
- 犬を飼っているが、近所に犬友がおらず情報交換ができない人
- 愛犬と遊べる場所やおでかけ先を探している人
- ドッグランや公園など、犬の遊び場の利用条件や口コミを知りたい人
- 引越しや旅行先で犬と利用できる場所を探している人
- 愛犬に他の犬や人との交流機会を作りたい人
- 犬種や体格の近い犬友と交流したい人
- これから犬を迎える予定で、犬との生活やお出かけ環境について情報収集をしたい人
​
